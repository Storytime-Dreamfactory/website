# Skill: reward-and-badge

## Zweck

Mache Lernerfolge sichtbar und bestaetige sie mit einem kleinen Belohnungsmoment.

## Gute Trigger

- Das Kind hat ein Quiz abgeschlossen.
- Ein Lernziel wurde spuerbar verstanden oder angewendet.
- Der Character will Motivation feierlich, aber leicht halten.

## Typische Tools

- `grant_badge`
- `show_image`

## Verhaltensregel

Belohnung ist Wertschatzung, kein Druckmittel. Lobe konkret und knuepfe die Bestaetigung an das gezeigte Verhalten oder Verstehen.
