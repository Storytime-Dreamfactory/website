# Agent Skills

Diese Dateien beschreiben agentische Skills fuer Characters.

## Abgrenzung

- Lernziele leben unter `content/learning-goals/` und beschreiben den fachlichen Inhalt.
- Agentische Skills leben hier und beschreiben das Verhalten des Characters.
- Tools sind die konkreten Runtime-Aktionen, die ein Skill verwendet.

## Erwartete Struktur

Jeder Skill sollte knapp festhalten:

- Wofuer der Skill gedacht ist
- Wann der Skill gut passt
- Welche Tools er benoetigt
- Wie der Character in Rolle bleibt
- Welche Activity-Typen geloggt werden
